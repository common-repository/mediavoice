Deliver native ads on Wordpress with the MediaVoice WordPress Plugin.

== Description ==

The MediaVoice WordPress Plugin lets you easily deploy your MediaVoice Script on your website, immediately readying your native ad units for content.

This plugin works alongside the MediaVoice Hub to display native advertisements on your WordPress site. An active account with MediaVoice is required to use this plugin.

For more information visit http://polar.me/mediavoice

Having trouble? Contact us for help at nativeads.support@polar.me

== Installation ==

Installation of MediaVoice to your WordPress site is simple.

To install from the wp-admin panel:

1. Login to your admin page at http://yoursite.com/wp-admin/
1. Navigate to "Plugins" -> "Add New menu item"
1. If you have already downloaded the MediaVoice plugin, click upload and
select the mediavoice.zip on your system
1. If you don't already have the .zip you can search for 'MediaVoice' in the
search box
1. Follow the on screen instructions to install and activate the MediaVoice
plugin

To install manually:

1. Download the MediaVoice plugin .zip file from the WordPress Plugin Directory
1. Unzip the file and upload to your web server's "/wp-content/plugins/" directory
1. Activate the MediaVoice plugin through your wp-admin "Plugins" section

Once installed and activated:

1. Navigate to the MediaVoice admin interface "Settings" -> "MediaVoice Plugin"
1. Read the instructions and check the "Enable MediaVoice Script" box
1. Paste your ad code from the MediaVoice bookmarklet to the "Ad Code" textarea
click "Save Changes"
